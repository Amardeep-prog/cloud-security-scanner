# 🔐 Cloud Security Scanner

A production-grade API & Cloud Security Scanner that detects vulnerabilities, misconfigurations, and cloud exposure risks.



## 🚀 Live Demo
👉 https://cloud-security-scanner.onrender.com/docs


## 🚀 Features

- 🔍 API Security Scanning (Headers, CORS, Response)
- ☁️ Cloud Detection (AWS S3, API Gateway)
- ⚡ Async Parallel Scanning Engine
- 🌐 Subdomain Discovery
- 🧪 API Fuzzing
- 📊 Risk Scoring (A+ → F)
- 📁 Scan History Dashboard
- 🔔 Alert System

---

## 🛠 Tech Stack

- FastAPI (Backend)
- React (Frontend)
- AWS (S3, DynamoDB)
- AsyncIO + httpx
